# 收费解锁测试报告

> 测试引擎：`config.py` → `coverage_service.py` → `payment.py`
> 收费门槛：commercialScore ≥ 85

---

## 一、收费判定逻辑

```
payment.py: create_payment()
  → Config.get_allow_payment()
    → 检查 ALLOW_PAYMENT 环境变量
    → 否则: coverage_service.should_allow_payment()
      → calculate_coverage()["commercialScore"] >= 85
```
