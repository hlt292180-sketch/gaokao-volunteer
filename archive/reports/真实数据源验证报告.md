**关键发现：与模拟数据的差异**

| 指标 | 模拟数据 | 真实数据 |
|------|----------|----------|
| 粒度 | 院校级(30专业/校) | **院校专业组级**(N个组/校) |
| 南大物理最低 | 627(冷门专业) | **661(07组/不限)** |
| 南航物理最低 | 554(冷门专业) | **655(04组/化学)** |
| 选科要求 | ❌ 无 | **✅ 化学/不限/化学+生物** |
| 南审物理最低 | ~470 | **562(05组/不限)** |

**结论：模拟数据严重偏离真实**。模拟分数系统性偏低 50-100 分，尤其低估了省属重点院校的真实录取线。

## 三、综合评估

### 分级获取策略

```
Level 1（立即可用）：院校专业组投档线
  来源：EOL/教育考试院 PDF → pdfplumber → CSV
  粒度：院校×专业组×选科要求
  覆盖：2023-2025 全部在苏招生院校

Level 2（2-3天获取）：专业录取分
  来源：各校招生网 → 爬虫 → CSV
  粒度：院校×专业×专业组×选科要求
  覆盖：55 所江苏院校可覆盖 80%

Level 3（持续补充）：分专业详细数据
  来源：多个数据源交叉验证
  粒度：含最高分/平均分/计划数
  覆盖：需人工审核匹配
```
