## 六、导入建议

### 优先级

1. **P0 - 立即导入**：2024年物理+历史（402行），覆盖最新一届考生
2. **P1 - 尽快导入**：2023年物理（240行），支持多年对比
3. **P2 - 补充后导入**：2023历史 + 2025全科

### 导入脚本（示例）

```sql
-- 清空模拟数据
DELETE FROM score_segments;

-- 导入真实数据（以 2024 物理为例）
LOAD DATA LOCAL INFILE 'real_data/score_segments_2024_physics.csv'
INTO TABLE score_segments
FIELDS TERMINATED BY ',' 
LINES TERMINATED BY '\n'
IGNORE 1 ROWS
(year, subject_type, score, @rank)
SET `rank` = @rank, province_id = 1;
```

### 插值逻辑兼容

当前 `score.py` 中的 `_interpolate_rank()` 和 `_interpolate_score()` 使用线性插值。真实数据导入后，插值精度将大幅提升（从5~20分采样 → 1分采样）。

但需要注意：**真实数据只在本科线以上有值**（如2024物理 462+）。本科线以下的分数查询需返回"无数据"而非插值。
