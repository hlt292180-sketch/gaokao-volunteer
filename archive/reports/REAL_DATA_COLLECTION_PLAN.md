```sql
CREATE TABLE admission_scores_archive LIKE admission_scores;
INSERT INTO admission_scores_archive SELECT * FROM admission_scores WHERE data_source='SIMULATED';
```

## 六、采集规范

### 必须包含字段
```
university_name  → 精确匹配 universities.name
year             → 2023/2024/2025
subject_type     → 物理类/历史类
major_group      → 07专业组/08专业组
subject_requirement → 化学/不限/化学+生物
min_score        → 投档最低分(整数)
min_rank         → 最低位次(从一分一段表换算)
```

### 验证规则
- min_rank 必须在对应年份一分一段表的有效范围内
- 同一专业组+同一年+同一科类不可重复
- 院校名称必须与 universities 表精确匹配
